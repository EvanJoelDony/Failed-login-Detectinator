## Overview
This project simulates a basic cybersecurity workflow: analyzing login attempt data to detect suspicious activity such as brute-force attacks. It combines **cybersecurity awareness** with **data analytics techniques** using Python.

## Features
- Counts successful vs. failed login attempts
- Detects suspicious IP addresses with repeated failures
- Visualizes login activity with charts
- Demonstrates log analysis and threat detection concepts

## Tech Stack
- Python 3
- pandas (data analysis)
- matplotlib (visualization)
